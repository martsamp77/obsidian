# Brand Graphics: Nano Banana Prompt Guide

## Before You Start

**Where to use Nano Banana:** Google AI Studio (aistudio.google.com), the Gemini app, or third-party interfaces like nanobanana.io or banana2.app. Google AI Studio gives you the most control.

**Important limitations:**

- Nano Banana outputs PNG/JPG only, not SVG. For a production logo you can scale to any size, you'll eventually need to vectorize the best result (use Vectorizer.ai, or Adobe Illustrator's Image Trace, or hire someone on Fiverr for $20 to convert your favorite PNG to SVG/AI format).
- Generate many variations. Plan on 20-30 generations per brand before you find one you love. Logos are subjective. Quantity leads to quality.
- Don't try to get the final logo in one prompt. Use a two-phase approach: Phase 1 is concept exploration (broad, many directions). Phase 2 is refinement (take the best concept, iterate on details).

**Prompt principles for logos:**

- Always specify "flat vector style" or "clean vector logo" to avoid photorealistic renders
- Always specify "white background" or "transparent background" for clean extraction
- Always specify "no mockup" to prevent it from placing the logo on a business card or sign
- Include the exact text you want rendered
- Specify what you do NOT want (gradients, 3D effects, photorealism, etc.)
- Keep prompts under 100 words for best results

## Brand 1: Dimension42 (Primary Brand)

### Concept Direction

D42 sits at the intersection of physics, AI, and the number 42 (Hitchhiker's Guide reference). The brand should feel technical, slightly sci-fi, intelligent, and premium. It should not feel playful, cartoonish, or overly corporate. Think: the logo you'd see on a black terminal screen or embossed on machined aluminum.

### Phase 1: Concept Exploration

Run each of these prompts 3-4 times to generate a spread of concepts.

**Prompt 1 — Wordmark focus:**

```
Design a modern tech company wordmark logo for "Dimension42". Clean sans-serif typography, the "42" should be visually integrated into the word "Dimension" as a cohesive unit. Minimal geometric style, monochrome palette (white on dark or dark on white). Flat vector logo, white background, no mockup, no tagline. The feel should be premium, technical, and intelligent.
```

**Prompt 2 — Icon + wordmark:**

```
Create a logo for "Dimension42", an AI and software company. The icon should abstractly reference a dimensional portal, tesseract, or geometric hypercube shape. Pair the icon with clean modern typography spelling "Dimension42". Color palette: deep navy blue and electric cyan. Flat vector style, white background, no mockup, no gradients, no 3D effects.
```

**Prompt 3 — Monogram/brand mark:**

```
Design a minimal geometric brand mark for "D42". The letters D, 4, and 2 should be fused into a single cohesive symbol. Think of it as an icon that works at 32px favicon size and at billboard scale. Style: angular, precise, technical. Colors: one accent color (electric blue or cyan) on white. Flat vector, no mockup, no text other than the D42 monogram.
```

**Prompt 4 — Physics-inspired:**

```
Create a logo for "Dimension42", a tech company inspired by physics and the number 42. The icon should subtly reference dimensional space, waveforms, or quantum concepts without being literal. Pair with clean modern sans-serif text "Dimension42". Monochrome with one accent color. Flat vector, minimal, white background, no mockup.
```

**Prompt 5 — Heavy metal edge (subtle):**

```
Design a tech company logo for "Dimension42" that has a subtle metallic, forged quality without being literal heavy metal band styling. Think precision-engineered, machined, angular typography with clean geometric lines. Dark background with silver/chrome colored text. Flat vector style, no mockup, professional and premium feeling.
```

### Phase 2: Refinement

Once you find a concept you like, use these refinement prompts. Upload the image you liked and add instructions:

```
Keep this exact logo concept but [make the typography bolder / reduce the icon size / change the blue to a deeper navy / make the "42" more prominent / simplify the icon / make it work better at small sizes / add more spacing between the icon and text].
```

### D42 Brand Mark (Icon Only)

The brand mark is the standalone icon that works without text. It goes on favicons, app icons, social media avatars, and watermarks.

```
Create a standalone brand mark icon for a tech company called "Dimension42". The icon should work at 32x32 pixels and at large scale. Abstract geometric shape that suggests dimensionality or a portal. No text in the icon. Single color plus white. Flat vector, square canvas, centered, white background.
```

## Production Checklist After Generation

Once you've selected your final logos from Nano Banana, do this:

- [ ] Save the highest resolution PNG of each logo (ideally 4K if Nano Banana Pro is available)
- [ ] Get each logo vectorized (SVG format) — use Vectorizer.ai or hire on Fiverr
- [ ] Create these variants for each brand logo:
    - [ ] Full color on white background
    - [ ] Full color on dark background
    - [ ] Single color (black) version
    - [ ] Single color (white) version
    - [ ] Icon only (no text)
    - [ ] Text only (no icon)
- [ ] Test each logo at these sizes:
    - [ ] 32x32 (favicon)
    - [ ] 80x80 (social media avatar display size)
    - [ ] 176x176 (YouTube channel icon)
    - [ ] 800x800 (social media profile upload)
    - [ ] 2560 wide (banner scale)
- [ ] Create a brand colors document with exact hex codes from your final logos
- [ ] Upload to Canva as a brand kit (colors, logos, fonts) for future asset creation
- [ ] Store all final assets in Obsidian at 10-Dimension42/Brand/ and 20-MartySampson/Brand/
