# Vault readme

This vault follows the numbered layout in [[Monday and Obsidian Organization]] (Obsidian section): `00-` transient, `10-`–`20-` brands, `30-`–`40-` company and infrastructure, `50-` personal, `60-` reference, `70-` archive, `90-` meta.

Daily notes: folder `01-Daily`, template `02-Templates/daily-note.md` (see `.obsidian/daily-notes.json`). The daily template uses **Templater**-style `{{date:...}}` tokens; install the Templater community plugin if you want those to expand automatically, or adapt the template to core template syntax.
