# PowerPoint Killer

## What Is It

A presentation tool that replaces PowerPoint/Keynote for the AI-native, zero-human operator. Built around the idea that presentations should be generated from structured content (outlines, data, briefs) by AI, not manually designed slide-by-slide.

## Why It Matters

PowerPoint is a 35-year-old paradigm built for manual design. The real work in presentations is structuring the argument and selecting the data — AI can handle both if given the right interface. A tool that generates professional decks from a brief or outline would save hours per presentation and could be a strong D42 product.

## Which Brand

- [x] Dimension42 — public-facing software product

## Rough Scope

Large (months) — this is a real product with AI generation, template system, and export capabilities.

## Notes

- Core loop: user writes an outline or brief → AI generates slide content + layout → user reviews and tweaks → export to PDF/PPTX
- Differentiation: focus on the zero-human operator use case (consulting decks, pitch decks, YouTube thumbnails/title cards)
- Technical approach: could be built on top of an existing presentation engine (Reveal.js, Marp) with an AI layer on top
- Revenue model: SaaS subscription or pay-per-deck under the D42 brand
